# El proyecto sin nombre

inicio: 24 de Agosto de 2023

objetivo: crear un generador de reportes automático que convierta archivos .csv en .html, cuyo resultado sea la salida de de aplicar el framework ydata-profiling, y que esté disponibilizado al público por medio de un endpoint de FastAPI

motivación: crear web apis que puedan interactuar con objetos tipo bytes (archivos)

producto final: que sea abierto al público
